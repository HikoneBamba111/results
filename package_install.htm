1    外部packagesインストール概要

Anacondaの標準packageのみでは我々の到達目標を達成できないため、必要な外部packageを適宜インストールする：

 

conda install conda-forge::ipython

conda install anaconda::tensorflow　　（今回、既にinstall済）

conda install conda-forge::matplotlib

conda install conda-forge::scikit-learn

conda install conda-forge::sympy

conda install conda-forge::pydot

conda install conda-forge::graphviz

conda install nrtrmp::japanize-matplotlib

conda install conda-forge::pandas　（追加）

conda install conda-forge::sympy_plot_backends（追加）

（pip install more_itertools）

 

今回、tensorflowを優先してインストールするために開設した個別環境をcondaからactivateして、その環境下で上記package群をinstallする。

以下、個別packageのインストール手順を１つ示すが、それ以外も同様。
